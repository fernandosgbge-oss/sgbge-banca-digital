# sgbge-banca-digital
SGBGE Digital es una plataforma de banca online (PWA) para Guinea Ecuatorial que centraliza acceso seguro, posición global (Corriente/Ahorro/DAT), movimientos y auditoría, transferencias CEMAC/SWIFT con conversor, pagos nacionales (SEGESA/Canal Sol/recargas), gestión de suscripciones, inversiones BVMAC y más...
